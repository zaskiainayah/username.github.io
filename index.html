About Me

Projects

Project Website

Sebelumnya, saya telah berhasil mengembangkan sejumlah situs web sederhana, termasuk salah satunya dalam sebuah proyek kelompok yang fokus pada pembuatan situs web tentang film. Dalam proyek ini, peran saya sebagai frontend developer memungkinkan saya untuk mengambil tanggung jawab dalam merancang tampilan dan antarmuka pengguna yang menarik.
